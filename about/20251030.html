📘 10/30学習テーマ：「ボタンで背景色ランダム変更アプリ」

目的：
JavaScriptでランダムな値（色）を生成し、DOM操作でHTMLの背景色を変える。
「イベント処理＋ランダム値＋スタイル操作」をまとめて体験します。

📂 フォルダ構成

css
project-folder/
├── index.html
├── style/
│   └── style.css
└── js/
    └── main.js

🧱 1. index.html（スマホ対応HTML）

📄 目的：
HTMLの構造を作り、ボタンとテキストを設置。
スマホでも見やすいようにレスポンシブ設定（meta viewport）も入れます。

html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>背景色ランダムチェンジャー</title>
  <link rel="stylesheet" href="style/style.css" />
</head>
<body>
  <main class="container">
    <h1>背景色チェンジャー 🎨</h1>
    <p id="colorName">現在の色：#ffffff</p>
    <button id="changeColorBtn">色を変える！</button>
  </main>

  <script src="js/main.js"></script>
</body>
</html>

✅ 解説

  • <meta name="viewport">：スマホで文字が拡大しないようにする設定。

  • <main class="container">：中央寄せのために使う。

  • <p id="colorName">：現在の色コードを表示する場所。

  • <button id="changeColorBtn">：クリックイベントの対象。

  • <script src="js/main.js">：JavaScriptファイルを読み込む。

🎨 2. style/style.css

📄 目的：
画面中央にボタンと文字を配置し、見やすいデザインを整える。

css
/* ====== 全体基本設定 ====== */
body {
  margin: 0;
  font-family: "Noto Sans JP", sans-serif;
  transition: background-color 0.5s ease;
  /* 色変化をなめらかに */
}

/* ====== 中央寄せコンテナ ====== */
.container {
  height: 100vh; /* 画面の高さ100% */
  display: flex;
  flex-direction: column;
  justify-content: center;
  align-items: center;
  text-align: center;
}

/* ====== タイトル ====== */
h1 {
  font-size: 1.5rem;
  margin-bottom: 1rem;
}

/* ====== 現在の色コード表示 ====== */
#colorName {
  font-size: 1.2rem;
  margin-bottom: 1.5rem;
}

/* ====== ボタンスタイル ====== */
#changeColorBtn {
  background-color: #333;
  color: #fff;
  border: none;
  border-radius: 8px;
  padding: 0.8rem 1.5rem;
  font-size: 1rem;
  cursor: pointer;
  transition: background-color 0.3s ease, transform 0.1s ease;
}

/* ====== ホバーと押したときの動き ====== */
#changeColorBtn:hover {
  background-color: #555;
}

#changeColorBtn:active {
  transform: scale(0.97);
}

✅ 解説

  • transition: background-color 0.5s ease;
    　背景色がスムーズに変化するようにする。

  • display: flex;
    　中央寄せを簡単にする便利な書き方。

  • #changeColorBtn:hover / :active
    　ユーザーが触ったときに軽い動きを付けて気持ちよく。

⚙️ 3. js/main.js

📄 目的：
クリックイベント → ランダム色生成 → 背景とテキスト更新、という処理を作る。

javascript
// =======================
// 背景色ランダムチェンジャー
// =======================

// 🎯 1. 要素を取得する
const changeColorBtn = document.getElementById("changeColorBtn");
const colorNameText = document.getElementById("colorName");

// 🎨 2. ランダムカラーを生成する関数
function getRandomColor() {
  // Math.random()：0〜1のランダム小数
  // それを 16777215 (16進数でFFFFFF) まで拡大して整数に
  const randomNumber = Math.floor(Math.random() * 16777215);
  // 16進数に変換して「#」をつける
  const randomColor = "#" + randomNumber.toString(16).padStart(6, "0");
  return randomColor;
}

// 🧩 3. ボタンをクリックしたときの処理
changeColorBtn.addEventListener("click", () => {
  const newColor = getRandomColor(); // 新しい色を作る
  document.body.style.backgroundColor = newColor; // 背景を変更
  colorNameText.textContent = `現在の色：${newColor}`; // テキスト更新
});

✅ 超丁寧なポイント解説

🔸 Math.random()

  • 0.0〜1.0 のランダムな数を返す（例：0.1234…）

  • 背景色（#RRGGBB）は 16進数の6桁なので、
    #ffffff（=10進で16777215）を上限にする。

🔸 .toString(16)

  • 10進数を16進数文字列に変換。

  • 例：255 → "ff"

🔸 .padStart(6, "0")

  • 6桁より短い場合、先頭に「0」を追加して桁を合わせる。

  • 例： "a3f" → "000a3f"

🔸 document.body.style.backgroundColor

  • HTML全体（body）の背景色を直接変更するDOM操作。

🔸 textContent

  • HTMLの中身のテキストを書き換える。

📱 スマホで見た目チェック

スマホでもきれいに見せるために、

  • フォントサイズは相対値（rem）で指定

  • 中央寄せをflexで自動調整

  • viewport指定済み

なので、横幅の小さい端末でもレイアウト崩れません。

💡 発展課題（余裕があれば）

  1. 文字色も自動で変わるようにする

    javascript
    document.body.style.color = newColor;

  2. 背景色を履歴として3件くらい表示する

    • ul要素を追加して、クリックするたびに履歴をappendChild。

  3. 「背景色をコピー」ボタンを追加

    • navigator.clipboard.writeText(newColor); を使う。
