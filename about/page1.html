ページリンクの作り方

　●フォルダ構造

　project/

　│

　├─ index.html

　├─ about/ ← 並列フォルダ

　│ └─ page.html ← 開きたい別ページ

　└─ contact/

　 └─ form.html

　●アンカーリンクを使う

　　①HTMLでリンクする方法

　　　1. 同じ階層のフォルダにあるHTMLを開く

　　　index.htmlからabout/page1.htmlへリンクする場合：<a href="about/page1.html">page1へ</a>

　　　index.htmlからabout/page1.htmlから新しいタブで開く場合：<a href="about/page1.html" target="_blank" rel="noopener noreferrer">page1へ</a>

　　　　　　target="_blank" → 新しいタブで開く

　　　　　　rel="noopener noreferrer" → セキュリティ対策（外部リンクや新しいタブ用）

　●ページ内リンク

　　　　<a href="#section1">1章へ移動</a>

　　　　同じページ内のリンク先

　　　　<h2 id="section1">1章

　　　　ページが長い場合に便利

　●メールリンク / 電話リンク

　　　　<a href="mailto:info@example.com">メールする</a>

　　　　<a href="tel:0123456789">電話する</a>

　　　　スマホではタップで即メール・電話アプリが開く

　● 別ページ側に戻るリンクを置く場合

　　　　about/page.htmlからindex.htmlに戻すには：<a href="../index.html">トップページに戻る</a>

　　　　../ は「1つ上の階層へ戻る」という意味

　　　トップページに戻る
