🧠 3. JavaScript（js/script.js）最終調整 ＋ 演出追加（25分）

▼ 問題データ（例）

javascript
const quiz = [
    {
        question: "日本で一番高い山は？",
        choices: ["富士山", "北岳", "奥穂高岳"],
        answer: 0
    },
    {
        question: "JavaScriptの略称は？",
        choices: ["JS", "JVS", "J-Script"],
        answer: 0
    },
    {
        question: "HTMLで正しいタグは？",
        choices: ["<div>", "<dv>", "<d>"],
        answer: 0
    }
];

▼ 必要な要素取得

javascript
const questionEl = document.getElementById("question");
const choiceBtns = document.querySelectorAll(".choice-btn");
const statusEl = document.getElementById("status");
const nextBtn = document.getElementById("next-btn");

let current = 0;
let score = 0;

▼ 問題を読み込む関数

javascript
function loadQuestion() {
    const q = quiz[current];
    questionEl.textContent = q.question;

    choiceBtns.forEach((btn, i) => {
        btn.textContent = q.choices[i];
        btn.disabled = false;
        btn.style.background = "#e3e3e3";
    });

    statusEl.textContent = "";
    nextBtn.style.display = "none";

    resetFadeIn();
}

▼ フェードインを再適用

javascript
function resetFadeIn() {
    const container = document.querySelector(".quiz-container");
    container.classList.remove("fade-in");

    setTimeout(() => {
        container.classList.add("fade-in");
    }, 10);
}

▼ 回答処理

javascript
choiceBtns.forEach(btn => {
    btn.addEventListener("click", function() {
        const index = Number(btn.dataset.index);
        const q = quiz[current];

        if (index === q.answer) {
            score++;
            statusEl.textContent = "正解！";
            btn.style.background = "#4caf50";
        } else {
            statusEl.textContent = "不正解…";
            btn.style.background = "#f44336";
        }

        choiceBtns.forEach(b => b.disabled = true);
        nextBtn.style.display = "block";
    });
});

▼ 次の問題へ

javascript
nextBtn.addEventListener("click", () => {
    current++;

    if (current < quiz.length) {
        loadQuestion();
    } else {
        showResult();
    }
});

▼ 結果表示

javascript
function showResult() {
    questionEl.textContent = `結果：${score} / ${quiz.length}`;
    document.getElementById("choices").style.display = "none";
    nextBtn.style.display = "none";
    statusEl.textContent = "おつかれさま！";
}

🐛 4. よくあるバグと修正チェックリスト（5分）

✔ フェードインがつかない

✔ 結果画面で選択肢が残る

→ choices の要素を display:none にする必要あり（上のコードで対応済み）
