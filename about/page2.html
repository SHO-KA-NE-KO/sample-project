WEB開発の基礎知識

第1章　WEBの世界を地図で見る

　1-1. HTMLって何者？

　HTMLは「HyperText Markup Language」の略

　一言でいうと “WEBページの設計図”

　WEBページの「見出し」「段落」「画像」「リンク」など、**構造（骨組み）**を作る言語。

　　📌 例え：

　　HTML＝家の骨組み

　　CSS＝壁紙・家具・色

　　JavaScript＝自動ドアや電気などの動く仕掛け

　

　1-2. WEBページの役割分担

　HTML → 文章や見出し・ボタンなどの「中身と構造」を作る

　CSS → 色・形・余白などの「見た目」を整える

　JavaScript → ボタンを押したら動く、データを読み込むなど「動きや機能」をつける

　

　2-1. HTMLはタグでできている

　タグ：< >で囲まれた命令。

　基本的に「開始タグ」と「終了タグ」のペアで使う。

　<p>こんにちは！</p>

　→ <p>は段落（paragraph）のタグ。</p>で終わる。

　

　2-2. HTMLの全体構造

　<!DOCTYPE html>

　<html lang="ja">

　<head>

　　　<meta charset="UTF-8">

　　　<title>ページのタイトル</title>

　</head>

　<body>

　　　<h1>見出し</h1>

　　　<p>ここに文章を書きます。</p>

　</body>

　</html>

　各パーツの役割

　<!DOCTYPE html> → HTML5を使う宣言

　<html>〜</html> → ページ全体を包む

　<head> → ページの設定情報（見えない部分）

　<meta charset="UTF-8"> → 文字化け防止（UTF-8の文字コード）

　<title> → ブラウザタブの名前

　<body> → 画面に表示される部分（本文）
